getRes

  • getRes<T>(name: string, type?: ResType): Res<T>

  • 获取资源对象

    Type parameters

    • T

    Parameters

    • name: string

      资源名 资源名为资源文件root中的相对路径，不带文件扩展名，使用/分隔目录，不区分大小写 资源名前缀为ext: http: https:的则当作外部资源加载 资源前缀为R:的则直接使用hash后的资源id加载

    • Optional type: ResType

      如果为外部资源则需要指定type，若不指定则当image来加载

    Returns Res<T>

    资源对象
