Static releaseUnusedTexture

  • releaseUnusedTexture(maxAge: number): void

  • Inherited from Texture.releaseUnusedTexture

    释放长期未使用的WGL纹理 在当前加载的纹理中，每帧未被使用的age会加1，被使用的age设为0

    Parameters

    • maxAge: number

      超过maxAge的纹理释放

setInterval(() => egl.Texture.releaseUnusedTexture(60), 1000);

    Returns void
