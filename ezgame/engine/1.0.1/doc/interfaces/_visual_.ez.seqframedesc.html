Interface SeqFrameDesc

动画序列帧数据定义

Hierarchy

  • SeqFrameDesc

Properties

  • count
    • 总帧数
  • fps
    • 动画fps
  • frames
    • 用逗号分隔的序列帧图片名 如果指定了frames则优先使用frames生成序列帧图片名字数组
  • from
    • 起始帧编号 使用from,count自动生成序列帧图片名字数组 from可以使用数字，也可以使用带前导0的字符串，在生成序列帧时会自动填充前导0的个数 例如{ prefix:"f", form:"008", count:4 } 会生成出 ["f008","f009","f010","f011"]
  • loop
    • 是否循环播放
  • prefix
    • 序列帧图片名前缀

Properties

Optional count

count: number

总帧数

Optional fps

fps: number

动画fps

Optional frames

frames: string

用逗号分隔的序列帧图片名 如果指定了frames则优先使用frames生成序列帧图片名字数组

Optional from

from: number | string

起始帧编号 使用from,count自动生成序列帧图片名字数组 from可以使用数字，也可以使用带前导0的字符串，在生成序列帧时会自动填充前导0的个数 例如{ prefix:"f", form:"008", count:4 } 会生成出 ["f008","f009","f010","f011"]

Optional loop

loop: boolean

是否循环播放

prefix

prefix: string

序列帧图片名前缀
